## 技术栈

TypeScript + Vue

## 使用

使用Gabor Patch训练你的眼睛，以提高你的视力， 用于弱视治疗。

## 使用方法

1. 下载代码
2. 运行 `npm run dev` 启动服务
3. 打开 `http://localhost:13142` 访问
4. 点击 `开始训练` 按钮
5. 按照提示完成训练

## 训练模式说明

### 1. 单图训练
- 操作要求：屏幕闪烁两次，一次有 Gabor 视标，另一次没有。
- 按键操作：
  - 如果第一次闪烁出现视标，点击鼠标左键或按左箭头键
  - 如果第二次闪烁出现视标，点击鼠标右键或按右箭头键

### 2. 三图训练  
- 操作要求：屏幕闪烁两次，一次有 3 个 Gabor 视标，另一次只有 2 个 Gabor 视标。（可能两次很接近，视标会接近融合）
- 按键操作：
  - 如果第一次闪烁出现 3 个 Gabor 视标，点击鼠标左键或按左箭头键
  - 如果第二次闪烁出现 2 个 Gabor 视标，点击鼠标右键或按右箭头键

### 3. 清晰图像训练
- 操作要求：屏幕闪烁两次，两次均有 Gabor 视标，一次较清晰，一次较淡。（对于弱视眼可能差异非常小）
- 按键操作：
  - 如果清晰的视标是第一次出现，点击鼠标左键或按左箭头键
  - 如果清晰的视标是第二次出现，点击鼠标右键或按右箭头键

### 4. 移图训练
- 操作要求：屏幕闪烁一次，出现 3 个 Gabor 视标，中间的视标可能往上下左右任意一个方向偏移。（对于弱视眼可能差异非常小）

- 按键操作：
  - 视标往左偏：点击鼠标左键或按左箭头键
  - 视标往右偏：点击鼠标右键或按右箭头键
  - 视标往上偏：将鼠标逆时针横放，点击鼠标右键，或按右箭头键
  - 视标往下偏：将鼠标逆时针横放，点击鼠标左键，或按左箭头键

### 5. 拥挤训练
- 操作要求：
  - 屏幕闪烁两次
  - 第一次显示一个由 E 字形成的阵列，其中中间有一个 E 字
  - 第二次显示一个类似的阵列，但中间为空白
- 按键操作：
  - 如果第一次闪烁中间有 E 字，点击鼠标左键或按左箭头键
  - 如果第二次闪烁中间有 E 字，点击鼠标右键或按右箭头键
- 说明：
  - 该训练模式类似于单图训练，但视标为 E 字阵列
  - 重点在于识别中间位置的变化

### 训练说明
- 训练单元：
  - 每次训练包含十一个单元
  - 每个单元有 50 次偏移判断
  - 每个单元开始前，点击鼠标中键或按向上箭头键开始训练
- 按键替换：
  - 鼠标左键可替换为左箭头键
  - 鼠标右键可替换为右箭头键
  - 鼠标中键可替换为向上箭头键
  - 进入下一单元使用向下箭头键
  - 可使用游戏手柄映射键盘或鼠标操作
- 其他说明：
  - 鼠标指针为红色放大版本，便于观察
  - 训练间隔需要 8 小时
  - 失败和错误会播放提示音，并短暂显示放大的正确图像，之后渐渐缩小，便于观察
  - 训练质量综合了正确率，反应时间，和稳定性（稳定的正确率和反应时间，跳动幅度过大会减小权重）
  - 难度主要体现在视标大小，视标清晰程度，闪烁间隔。对于一个单元的训练，视标大小不变，闪烁间隔不变，视标清晰程度会逐渐减小，间距或者偏移程度会更小，以此增大难度。随着训练程度增加，视标大小会减小，清晰度会减弱。闪烁间隔会减小。
  - 训练数据以 csv 文件保存，训练结束后可查看训练数据，可生成训练报告和进步折线图。

### 环境要求
- 在暗室进行训练，确保房间内除电脑屏幕外无其他光源
- 训练者距离电脑屏幕为 1.5 米

### 注意事项
- 疲劳处理：出现视疲劳时可短暂休息，待症状缓解后继续
- 操作建议：可使用游戏手柄映射键盘或鼠标操作
- 开始前请校准屏幕颜色和尺寸

## 训练编排

第一二次为适应性训练，提供多种类型，为五个单元，可点击按钮跳过。

正式训练中：
+ 第五个拥挤训练为独立的训练
+ 初期（前二十次）以单图训练，三图训练训练为主
+ 中期（二十到三十次）以清晰图像训练，移图训练为主
+ 后期（三十次以后）多个训练类型穿插
+ 在训练过程中，难度会循序渐进，根据训练质量，调整难度
